# 📋 Инструкция: Как вставить SQL запросы в pgAdmin

## 🔧 Пошаговая инструкция

### Шаг 1: Откройте pgAdmin

1. Откройте браузер и перейдите на: **http://localhost:5050** (или ваш адрес pgAdmin)
2. Введите логин и пароль для входа в pgAdmin

---

### Шаг 2: Подключитесь к серверу PostgreSQL

1. В левом меню найдите **Servers**
2. Разверните дерево и найдите ваш сервер PostgreSQL (обычно **PostgreSQL 18**)
3. Нажмите правой кнопкой мыши на название сервера и выберите **Properties**
4. Вернитесь обратно

---

### Шаг 3: Откройте базу данных CRMbank

1. Разверните ваш сервер PostgreSQL
2. Найдите папку **Databases**
3. Двойной клик на **CRMbank** (или ваша база данных)

---

### Шаг 4: Откройте Query Tool (Редактор запросов)

**Вариант 1:** Нажмите на иконку молнии ⚡ в верхнем меню  
**Вариант 2:** Right-click на базу данных → **Query Tool**  
**Вариант 3:** Tools → Query Tool

Откроется новое окно с редактором SQL запросов

---

### Шаг 5: Скопируйте SQL скрипт

1. Откройте файл `SQL_FOR_PGADMIN.sql` из проекта CRM
2. Нажмите **Ctrl+A** чтобы выделить весь текст
3. Нажмите **Ctrl+C** чтобы скопировать

---

### Шаг 6: Вставьте в pgAdmin

1. Нажмите в окне Query Tool (чтобы установить фокус)
2. Нажмите **Ctrl+V** чтобы вставить код
3. Весь SQL скрипт будет вставлен в редактор

**Результат должен выглядеть так:**
```sql
-- ============================================================================
-- SQL СКРИПТЫ ДЛЯ СОЗДАНИЯ ТАБЛИЦ CRM СИСТЕМЫ
-- ...
CREATE TABLE status (
    id SERIAL PRIMARY KEY,
    ...
)
...
```

---

### Шаг 7: Выполните запрос

**Вариант 1 (Выполнить всё):**
- Нажмите кнопку ▶️ **Execute/Refresh** в верхнем меню (или **F5**)

**Вариант 2 (Выполнить по частям):**
- Выделите нужные строки (Ctrl+A для всех)
- Нажмите F5

**Вариант 3 (Через меню):**
- Query → Execute (или Query → Execute/Refresh)

---

### Шаг 8: Проверьте результат

Внизу окна появится вкладка **Data Output** с сообщением:

```
Query executed successfully - No results
```

Это означает, что все таблицы созданы ✅

## 📝 Альтернативный способ: Импорт файла

Если вы хотите загрузить SQL файл напрямую:

1. В pgAdmin нажмите **File** → **Open** (или Ctrl+O)
2. Выберите файл `SQL_FOR_PGADMIN.sql`
3. Нажмите **Execute**

---

## ⚠️ Если возникла ошибка

### Ошибка: "Database CRMbank does not exist"

1. Сначала создайте базу данных:
   ```sql
   CREATE DATABASE CRMbank;
   ```
2. Затем подключитесь к ней и вставьте скрипт

### Ошибка: "Table already exists"

Это означает, что таблицы уже созданы. Выполните:
```sql
-- Удалить все таблицы (ВНИМАНИЕ: данные будут потеряны!)
DROP TABLE IF EXISTS ticket_history CASCADE;
DROP TABLE IF EXISTS attachment CASCADE;
DROP TABLE IF EXISTS message CASCADE;
DROP TABLE IF EXISTS ticket CASCADE;
DROP TABLE IF EXISTS operator CASCADE;
DROP TABLE IF EXISTS client CASCADE;
DROP TABLE IF EXISTS channel CASCADE;
DROP TABLE IF EXISTS priority CASCADE;
DROP TABLE IF EXISTS status CASCADE;

-- Затем вставьте весь скрипт снова
```

### Ошибка: "Database is in use"

Закройте все Query Tool окна для этой базы данных и повторите.
